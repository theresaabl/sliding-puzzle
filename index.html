Sliding Puzzle

  • This website provides a classic sliding puzzle game, which can be played with puzzle sizes between 2 x 2 and .
  • The goal is to order the tiles in ascending order (1, 2, 3, ...) with the empty tile in the right bottom corner.
  • To move a tile 1 click on it (or touch it for mobile devices).
  • To start a new game click the New Game button.
  • To reshuffle the puzzle click the Reshuffle button.
  • The number of moves and the time needed to solve the puzzle are displayed and a leaderboard is available when clicking on the - icon.
